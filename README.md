# Sentiment Analysis using Logistic Regression (From Scratch)

This project implements a sentiment analysis model using Logistic Regression built completely from scratch without using sklearn.

## Dataset
AG News Dataset (converted into binary sentiment classification)

## ⚙️ Features
- Text preprocessing (tokenization, stopwords removal, stemming)
- Frequency dictionary
- Feature extraction ([1, pos_count, neg_count])
- Logistic regression (sigmoid, cost function, gradient descent)

## Results
- Accuracy: ~85%
- Includes error analysis of misclassified samples
